# パフォーマンス

## 基本方針

サイトは軽量にしてください。

## 優先すること

- 不要な外部ライブラリを使わない
- 画像を無駄に大きくしない
- CSSとJavaScriptを必要最小限にする
- 初期表示を重くしない
- アニメーションを入れすぎない

## 画像

大きすぎる画像は避けてください。

サムネイル画像は、表示サイズに対して極端に大きくならないようにしてください。

## JavaScript

初期段階では、JavaScriptはUI補助程度にしてください。

複雑な状態管理や外部API連携は不要です。

## CSS

CSSは読みやすく保ちつつ、不要な重複を避けてください。

## 外部リソース

外部CDNや外部フォントは、必要な場合のみ使ってください。

初期段階では、システムフォントを優先します。
