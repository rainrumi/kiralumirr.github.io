## 編集方法

- トップページ: `index.html`
- スタイル: `css/style.css`
- JavaScript: `js/main.js`
- 画像: `assets/images/`
- アイコン: `assets/icons/`
- フォント: `assets/fonts/`

未確定の文章やリンクは、`TODO` が付いた箇所を編集してください。

## 画像の差し替え方法

画像は `assets/images/` に追加します。ファイル名は英数字とハイフンを使い、HTMLの `src` と `alt` をあわせて更新してください。

## GitHub Pagesの公開メモ

GitHub Pagesでは、リポジトリ直下の `index.html` を公開入口にできます。サーバーサイド処理やビルド手順は不要です。
