# 独自ドメイン設定メモ

## 基本方針

初期段階では独自ドメインを設定しません。

まずはGitHub Pages標準URLで表示確認を行います。

## 独自ドメインを設定する前の条件

以下が確認できてから独自ドメインを設定してください。

- `index.html` が表示できる
- CSSが反映されている
- JavaScriptエラーがない
- 画像パスが壊れていない
- スマホ表示が大きく崩れていない
- 主要リンクが動作する

## CNAMEファイル

独自ドメインを設定する場合、必要に応じてリポジトリ直下に `CNAME` ファイルを置きます。

例：

```text
example.com
```

## 注意点

DNS反映には時間がかかる場合があります。  
サイト側の問題とDNS側の問題を切り分けるため、独自ドメイン設定前にGitHub Pages標準URLでの表示確認を済ませてください。
