# アクセシビリティ

## 基本方針

最低限のアクセシビリティを守ってください。

## 必須ルール

- セマンティックHTMLを使う
- 見出し階層を崩さない
- 意味のある画像には `alt` を付ける
- 装飾画像は `alt=""` にする
- フォーカス表示を消さない
- ボタンには意味のあるテキストを入れる
- リンクテキストはリンク先が分かる文言にする
- 文字色と背景色のコントラストを確保する
- 色だけで意味を伝えない

## ボタン

ボタンには、機能が分かるテキストを含めてください。

良い例：

```html
<button type="button">メニューを開く</button>
```

悪い例：

```html
<button type="button">☰</button>
```

アイコンだけのボタンにする場合は、`aria-label` などを使って意味を補ってください。

## キーボード操作

キーボード操作を妨げないでください。

特に、フォーカスアウトラインを安易に消さないでください。
