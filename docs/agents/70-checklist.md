# 変更時のチェックリスト

## 基本チェック

作業完了前に、以下を確認してください。

- `index.html` がリポジトリ直下にある
- CSSのパスが正しい
- JavaScriptのパスが正しい
- GitHub Pagesで公開できる静的構成になっている
- サーバーサイド処理を追加していない
- npmやフレームワークを勝手に導入していない

## 表示チェック

- PCで大きく崩れていない
- スマホで大きく崩れていない
- 主要セクションが読める
- 画像パスが壊れていない
- リンクが極端に分かりにくくない

## コンテンツチェック

- 仮テキストには `TODO` が付いている
- 架空の正式プロフィールを追加していない
- 架空の作品情報を本物のように書いていない
- 実在するSNSやメールを勝手に書いていない

## アクセシビリティチェック

- 見出し階層が大きく崩れていない
- 意味のある画像に `alt` がある
- フォーカス表示を消していない
- ボタンやリンクの意味が分かる

## ファイルチェック

- 日本語ファイル名を使っていない
- 画像は `assets/images/` にある
- CSSは `css/` にある
- JavaScriptは `js/` にある
- ドキュメントは `docs/` にある

## READMEチェック

- 編集方法が書かれている
- 公開URL欄がある
- 使用技術が書かれている
- TODO欄がある
