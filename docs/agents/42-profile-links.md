# プロフィール・リンク欄

## Profileセクション

プロフィール欄には、以下を配置できる構造にしてください。

- 名前または活動名
- 肩書き
- 短い紹介文
- アイコン画像
- 活動ジャンル
- SNSリンク

未確定の情報は `TODO` にしてください。

## Linksセクション

外部リンク欄には、以下のようなサービスへのリンクを置ける構造にしてください。

- X
- YouTube
- GitHub
- BOOTH
- Ci-en
- itch.io
- Steam
- note
- メール

## リンク表記

リンクテキストは、リンク先の意味が分かるものにしてください。

良い例：

```html
<a href="#">Xで最新情報を見る</a>
<a href="#">GitHubでコードを見る</a>
```

悪い例：

```html
<a href="#">こちら</a>
```

## 注意点

実在するSNSアカウントやメールアドレスを勝手に書かないでください。

ユーザーから与えられていないリンクは `#` または `TODO` として扱ってください。
