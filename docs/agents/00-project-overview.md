# プロジェクト概要

## 目的

このリポジトリは、GitHub Pagesで公開する静的な個人サイト・ポートフォリオサイトを管理するためのものです。

想定するサイト種別は以下です。

- 個人プロフィールサイト
- ゲーム制作者サイト
- サークルサイト
- 作品ポートフォリオ
- 活動リンク集
- 更新履歴を掲載する小規模サイト

## 前提

このサイトは、静的ファイルのみで構成します。

使用するもの：

- HTML
- CSS
- Vanilla JavaScript
- Markdown
- 静的画像ファイル

使用しないもの：

- サーバーサイド処理
- データベース
- ログイン機能
- CMS
- 管理画面

## コンテンツ方針

サイト本文は、ユーザーが後から自分で編集します。

AIエージェントは、ユーザーから明示されていない情報を事実のように書かないでください。

未確定の文章、名前、作品名、リンク、日付などは `TODO` として扱います。
