# README作成ルール

## 目的

`README.md` は、このサイトを後から編集・公開しやすくするための説明書です。

## READMEに含める内容

以下を含めてください。

- サイト名
- サイト概要
- 公開URL
- 使用技術
- ディレクトリ構成
- 編集方法
- 画像の差し替え方法
- GitHub Pagesの公開メモ
- 今後のTODO

## 例

```md
# TODO: サイト名

GitHub Pagesで公開する個人サイトです。

## 公開URL

TODO: GitHub PagesのURLを入れる

## 使用技術

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## 編集方法

- トップページ: `index.html`
- スタイル: `css/style.css`
- JavaScript: `js/main.js`
- 画像: `assets/images/`

## TODO

- プロフィール文を追加する
- 作品情報を追加する
- OGP画像を作成する
- 独自ドメインを設定する
```

## 注意点

READMEにも、ユーザーが明示していない正式情報を勝手に書かないでください。
