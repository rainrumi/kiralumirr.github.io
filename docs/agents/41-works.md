# 作品一覧セクション

## 目的

`Works` セクションは、制作物を一覧表示するための領域です。

## 表示形式

作品はカード形式で表示してください。

各カードに含めるもの：

- 作品タイトル
- サムネイル画像
- 短い説明文
- ジャンル
- 公開状態
- 詳細リンク
- 外部リンク

## HTML例

```html
<article class="work-card">
  <img src="assets/images/work-placeholder.png" alt="TODO: 作品サムネイル">
  <h3>TODO: 作品タイトル</h3>
  <p>TODO: 作品説明文を入れる。</p>
  <a href="#">TODO: 詳細を見る</a>
</article>
```

## 注意点

ユーザーが明示していない作品名、公開日、説明文を事実のように書かないでください。

未確定情報はすべて `TODO` にしてください。

## 追加しやすい構造

カードをコピーして増やせるように、HTML構造を単純にしてください。

初期段階では、作品データのJSON化や外部読み込みは不要です。
