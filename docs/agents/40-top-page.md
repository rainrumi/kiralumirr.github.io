# トップページ構成

## 基本セクション

トップページには、以下のセクションを用意してください。

```text
Header
Hero
Profile
Works
Links
Updates
Footer
```

## Header

含めるもの：

- サイト名
- トップページへのリンク
- グローバルナビゲーション
- スマホ用メニュー

ナビゲーション例：

```text
Home
Profile
Works
Links
Contact
```

## Hero

含めるもの：

- メインタイトル
- サブタイトル
- 短い説明文
- メインビジュアルまたはアイコン
- 主要リンクボタン

未確定の文言は `TODO` にしてください。

## Profile

含めるもの：

- 名前または活動名
- 肩書き
- 短い紹介文
- 活動ジャンル
- SNSリンク

## Works

作品カードを並べるセクションです。  
詳細は `docs/agents/41-works.md` を参照してください。

## Links

外部サービスへのリンクをまとめるセクションです。  
詳細は `docs/agents/42-profile-links.md` を参照してください。

## Updates

簡単な更新履歴を表示します。

例：

```text
2026-06-01 TODO: サイト公開
```

## Footer

含めるもの：

- コピーライト
- サイト名
- トップへ戻るリンク
- 必要に応じたSNSリンク
