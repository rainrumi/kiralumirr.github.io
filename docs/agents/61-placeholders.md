# TODO・仮テキスト運用

## 基本方針

未確定の情報は、必ず `TODO` として扱ってください。

## TODOにするもの

- サイトタイトル
- プロフィール文
- 活動名
- 作品名
- 作品説明
- SNSリンク
- メールアドレス
- 公開日
- 更新履歴
- OGP画像
- 画像のalt文

## 良い例

```html
<h1>TODO: サイトタイトル</h1>
<p>TODO: ここにサイト説明文を入れる。</p>
```

```html
<a href="#">TODO: Xのリンクを入れる</a>
```

## 悪い例

```html
<h1>Mirai Works</h1>
<p>個人ゲーム制作者として活動しています。</p>
```

ユーザーが明示していない限り、実在しそうな情報を勝手に作らないでください。

## コメント

差し替え箇所が分かりやすいように、必要に応じてコメントを入れてください。

例：

```html
<!-- TODO: 正式なプロフィール文に差し替える -->
<p>TODO: ここにプロフィール文を入れる。</p>
```

ただし、コメントを書きすぎてHTMLが読みにくくならないようにしてください。
